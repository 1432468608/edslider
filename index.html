How to use?

Como usar?

1º - Call jQuery, edSlider plugin and edSlider css inside <HEAD> tag

1º - Importe o jQuery, o plugin e o css dentro da tag <HEAD>

<link rel="stylesheet" href="your-css-directory/edslider.css"> <script src="http://code.jquery.com/jquery-latest.min.js"></script> <script src="your-js-directory/edslider-1.2.js"></script>

2º - When document is ready, initialize the plugin (still in <HEAD> tag)

2º - Ao carregar o documento, inicialize o plugin (ainda dentro da tag <HEAD>)

<script> $(document).ready(function(){ $('.mySlideshow').edslider({ width:'960', height:'400' }); }); </script>

3º - Inside <BODY> tag, make the html structure like this:

3º - Dentro da tag <BODY> monte a estrutura html assim:

<ul class="mySlideshow"> <li>Yor stuff here!</li> <li>Yor stuff here!</li> <li>Yor stuff here!</li> </ul>
<ul class="meuSlideshow"> <li>Suas coisas aqui!</li> <li>Suas coisas aqui!</li> <li>Suas coisas aqui!</li> </ul>

Options

Opções

Option Default Value Description
Opção Valor padrão Descrição
width 960 Sets the width of the slider Define uma largura fixa à janela
height 400 Sets the height of the slider Define uma altura fixa à janela
position 1 Sets the start position. It must be between 1 and the <li> lenght. Define a posição inicial. Deve estar entre 1 e número de tags <li>.
interval 5000 Sets the time interval (in milliseconds) between slides. Define o intervalo de tempo (em milisegundos) entre os slides.
duration 500 Sets the speed (in milliseconds) of the animations. Define a velocidade (em milisegundos) da animação.
animation true Enable/disable animation. Hebilita/desabilita a animação.
paginator true Enable/disable the paginator (boolean value: true or false). Hebilita/desabilita a paginação (valor booleano: true ou false).
navigator true Enable/disable the "next" and "prev" navigation buttons (boolean value: true or false). Hebilita/desabilita a navegação para avançar ou voltar (valor booleano: true ou false).
progress true Enable/disable the interval progress bar (boolean value: true or false). Hebilita/desabilita a barra de progresso do intervalo (valor booleano: true ou false).
loadImgSrc 'images/load.gif' Sets the path of the loading image. Define o caminho da imagem de carregamento.
skin 'edslider' Sets the name of the skin. Use the edslider.css file to create your own skin. Define o nome do skin. Utilize o arquivo edslider.css como base para criar o seu próprio skin
